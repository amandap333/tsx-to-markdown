# FormReadOnlyField

## Usage
`import { FormReadOnlyField } from 'bscs-design-system/core'`

<FormReadOnlyField
  label='Read-only Field'
  name='Read-only Field'
  placeholder='Placeholder Text'
/>

### List of Props
#### Required
1. label
  - must have unique label

2. name
  - must have unique name

3. placeholder
   - Set the placeholder text you want

4. ...other

#### Optional
