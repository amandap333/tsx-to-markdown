# FormRadioField

## Usage

```
 <FormRadioField
  button={true}
  labels={['radio1', 'radio2']}
  setValue={(value) => value}
  title='Form Radio Field'
/>
```

### List of Props

#### Required

1. labels
2. setValue

#### Optional

1. button
   - default: false
2. inline
   - default: false
3. visible
   - default: true
4. ...other
